How to Use the Abacus

  • Click on beads to move them toward the beam
  • Upper beads (blue) represent a value of 5 each
  • Lower beads (red) represent a value of 1 each
  • The current value is calculated based on bead positions
  • Each rod represents a different place value (ones, tens, hundreds, thousands)
